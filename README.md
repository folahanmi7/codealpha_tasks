# Basic Network Sniffer in Python

A beginner-friendly project that demonstrates how to build a **simple network packet sniffer** using Python and **Scapy**. This project helps you understand how network traffic flows, how packets are captured, and how to extract useful information such as IP addresses and transport protocols.

## Project Overview

This project captures live network traffic from your local machine and displays:

- **Source IP address**
- **Destination IP address**
- **Transport protocol** (TCP / UDP / Other)

##  Objectives

- Understand what a **network sniffer** is  
- Learn how packets are captured  
- Identify **IP, TCP, and UDP** protocols  
- Practice Python with **Scapy**
